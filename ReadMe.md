## Instrucciones de uso

1- Clonar el repositorio

2- Credenciales de los `.env.local`

3- Backend:

```bash
   cd backend
   npm install
   npm run dev
```

4- Frontend:

```bash
   cd frontend
   npm install
   npm run dev
```
